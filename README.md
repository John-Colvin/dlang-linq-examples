## Installing and running

Dependencies to run these examples -- `dmd`, the reference D compiler  and `dub` -- the D pacakge manager:

On a Mac with [Homebrew](https://brew.sh) installed:

```
brew install dmd
```

Running examples:

```
dub
```

Running tests:

```
dub test
```
